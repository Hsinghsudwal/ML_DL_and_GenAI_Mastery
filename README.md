# 🚀 ML_DL_and_GenAI_Mastery

Every project is hands-on, practical, and portfolio-ready—designed to push me from beginner/intermediate to an advanced AI builder.

## 📖 About This Repository

This repo documents my journey through the 21 Projects, 21 Days: ML, Deep Learning & GenAI challenge hosted by GeeksforGeeks Live.

For me, this is more than just a course—it's a personal challenge to stay consistent, build daily, and sharpen my skills across Machine Learning, Deep Learning, and Generative AI.

Instead of just learning theory, I’m building real projects every day. Some projects revisit fundamentals, while others push into more advanced ideas like transformers, AI agents, and automation systems.

The goal is simple: learn by doing, every single day.

## 🧠 What I’m Learning

Across these 21 projects, I’m working on:

- Machine Learning: Supervised & unsupervised learning fundamentals
- Deep learning & neural networks
- Time series modeling & forecasting
- Anomaly detection systems
- Data engineering workflows
- Machine learning & deep learning pipelines
- Transformer architectures and model implementation
- Reinforcement learning with reward design
- Systems: Generative AI, Chatbot, End-to-end AI system design and AI agent development
- Real-world applications & deployment thinking
